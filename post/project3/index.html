Project 3: Automatically Generate Email Message (Simon & Schuster)

Overview

  • Efficiency tool developed to reduce redundant work
  • Includes six email templates that are automatically customized based on the assignment type, task completed, etc.
  • Deployed from a shared spreadsheet and able to be used by multiple people at once

Documentation

Select the entire row for a book and click on the colophon in the upper left to generate an auto-populated email draft. Select the text using your cursor, and copy the text using your keyboard (Ctrl/Command+C).
